you can either send a template using the `$template send` command

<Frame>
  <CodeGroup>

  ```text command
  $template send (collection) (channel) (name/uuid)
  ```

  </CodeGroup>

or by clicking send in the collection menu. after selecting a template, you're then asked to select a channel. you get the option to send it as the **bot** or as a **webhook**. if you want to edit an existing message, you can click **edi**t.

<Frame>

# webhooks

you can customize your webhook in this menu. variables like `{server_icon}` will work for the avatar. if you don't choose a name or an avatar, it uses kuyo's. you can also select from a list of **bot created webhooks** in that channel.
